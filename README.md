## 项目概述
本项目实现了网站内容管理功能，主要包括栏目、文章、单页、轮播图、推荐位等。

## 技术框架
- python3.6
- Django3.0
- django-mdeditor0.1.16

## 目录结构
- apps（应用）
- docs（文档）
- loncol_cms（配置）
- static（自定义静态文件）
- static_collect（静态文件收集）
- templates（模板文件）
- uploads（上传文件）
- venv（虚拟环境）
- README.md（帮助文件）
- VERSION.md（版本文件）
- requirements.txt（依赖文件）
